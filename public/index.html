API

ITEM

/api/items
METHOD ROUTE PARAMETERS REQUEST RESPONSE DESCRIPTION
GET
/api/items/getAll/:type type: client / inventory --- ARRAY of JSON Objects
  • Returns an array of items.
  • If client, excludes hidden items, contains item_id, item_name, category, tags, thumbnail.
  • If inventory, includes hidden items, contains all but item_description, thumbnail, images.
  • If empty, returns an error message.
/api/items/getOne/:type?id=xxx>

type: client / inventory

id: item_id

 --- JSON Object
  • Returns an item object.
  • If client, excludes hidden items, contains item_id, item_name,item_description, item_price, images, category, tags.
  • If inventory, includes hidden items, contains all attributes.
  • If item not found, returns an error message.
POST
/api/items/postOne --- JSON Object JSON Object
  • Send new item in JSON format.
  • If values of attributes are not specified, values are set to their default values.
  • Returns the new item if successful.
  • Returns an error message if unsuccessful.
/api/items/editOne?id=xxx id: item_id JSON Object {attribute: value, ...} JSON Object
  • Edits values of item with the new values sent as JSON Object.
  • Returns the new item.
  • Returns an error message if unsuccessful.
/api/items/disableOne?id=xxx id: item_id --- JSON Object {item, msg}
  • Sets item isHidden to true
  • Returns the item and feedback message if successful
  • Returns an error message if unsuccessful.
/api/items/enableOne?id=xxx id: item_id --- JSON Object {item, msg}
  • Sets item isHidden to false
  • Returns the item and feedback message if successful
  • Returns an error message if unsuccessful.
DELETE
/api/items/deleteOne?id=xxx id: item_id --- JSON Object {item, msg}
  • Permanently deletes item from the database.
  • If successful, returns the deleted item and a successful message.
  • If unsuccessful, returns a message.
